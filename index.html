GitHub Pages

GitHub pages son sitios web estaticos que nos ayudan a aprender facilmente.

¿Qué es GitHub Pages?

GitHub Pages es un servicio de alojamiento gratuito que permite publicar sitios web estáticos directamente desde un repositorio de GitHub.

Pasos para Desplegar con GitHub Pages

  1. Crea o sube tu proyecto a un repositorio en GitHub.
  2. Ve a "Settings" y selecciona "Pages".
  3. Elige la rama y carpeta raíz, y guarda los cambios.
  4. Tu sitio estará disponible en una URL similar a: `https://tu-usuario.github.io/nombre-del-repositorio/`.
